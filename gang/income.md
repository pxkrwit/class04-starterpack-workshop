# Cookie Run classic!!

#### เนื่องจากกระแสช่วงนี้ cookie run กำลังมาแรง มีผู้เล่นจำนวนมากที่สนใจและเริ่มเล่น เราจึงต้องอาศัยกระแสในช่วงนี้เพื่อทำรายได้ วิธีการหาเงินจาก cookie run คือ 
- **รับรีแอคจนกว่าได้สัตว์เลี้ยง สมบัติ หรือแม้กระทั่งคุกกี้ ตามที่ผู้จ้างต้องการ**
- **รับฟาร์มเงินรวมไปถึงฟาร์มเงินเพิ่มอัพเลเวลของสิ่งต่าง ๆ ที่ผู้จ้างได้มอบหมาย**
- **รับเติม diamond และ coin**
- **รับปั๊มสกอร์เยลลี่**
- **รวมไปถึงรับซื้อ-ขายไอดี**
